# CuddlyWombat
A restaurant management system that is capable of doing the following things
- Making Table Reservations
- Taking Orders
- Changing Orders
- Cancelling Orders
- Taking Payments
- Generating Sales Statistics
- Displaying Takeaway Menu
- Record Details for:
  - Reservations
  - Items
  - Menu
  - Receipts
  - Orders
  - Payments
  - Users
- Create Receipts
- Updating Online Menus
